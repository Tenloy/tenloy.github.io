еҺҹж–Үең°еқҖ иҜ‘иҮӘгҖҠiOS Core Animation Advanced TechniquesгҖӢ

еҠЁз”»жҳҜCore Animationеә“дёҖдёӘйқһеёёжҳҫи‘—зҡ„зү№жҖ§гҖӮиҝҷдёҖз« жҲ‘д»¬жқҘзңӢзңӢе®ғжҳҜжҖҺд№Ҳе·ҘдҪңзҡ„гҖӮе…·дҪ“жқҘиҜҙпјҢжҲ‘д»¬е…ҲжқҘи®Ёи®әжЎҶжһ¶иҮӘеҠЁе®һзҺ°зҡ„йҡҗејҸеҠЁз”»пјҲйҷӨйқһдҪ жҳҺзЎ®зҰҒз”ЁдәҶиҝҷдёӘеҠҹиғҪпјүгҖӮ

дёҖгҖҒеҸҜеҠЁз”»гҖҒдәӢеҠЎдёҺRunLoop

1.1 еҸҜеҠЁз”»зҡ„еӣҫеұӮеұһжҖ§

Core AnimationеҹәдәҺдёҖдёӘеҒҮи®ҫпјҢиҜҙеұҸе№•дёҠзҡ„д»»дҪ•дёңиҘҝйғҪеҸҜд»ҘпјҲжҲ–иҖ…еҸҜиғҪпјүеҒҡеҠЁз”»гҖӮдҪ е№¶дёҚйңҖиҰҒеңЁCore AnimationдёӯжүӢеҠЁжү“ејҖеҠЁз”»пјҢдҪҶжҳҜдҪ йңҖиҰҒжҳҺзЎ®ең°е…ій—ӯе®ғпјҢеҗҰеҲҷе®ғдјҡдёҖзӣҙеӯҳеңЁгҖӮ

еҪ“дҪ ж”№еҸҳCALayerдёҖдёӘеҸҜеҒҡеҠЁз”»зҡ„еұһжҖ§ж—¶пјҢиҝҷдёӘж”№еҸҳе№¶дёҚдјҡз«ӢеҲ»еңЁеұҸе№•дёҠдҪ“зҺ°еҮәжқҘгҖӮзӣёеҸҚпјҢиҜҘеұһжҖ§дјҡд»Һе…ҲеүҚзҡ„еҖје№іж»‘иҝҮжёЎеҲ°ж–°зҡ„еҖјгҖӮиҝҷдёҖеҲҮйғҪжҳҜй»ҳи®Өзҡ„иЎҢдёәпјҢдҪ дёҚйңҖиҰҒеҒҡйўқеӨ–зҡ„ж“ҚдҪң вҖ”вҖ” иҝҷе°ұжҳҜеҸҜеҠЁз”»зҡ„(йҡҗејҸеҠЁз”»)гҖӮ

еҸҜеҠЁз”»зҡ„(animatable)пјҡеҪ“ж”№еҸҳж—¶пјҢдјҡи§ҰеҸ‘дёҖдёӘд»Һж—§зҡ„еҖјиҝҮжёЎеҲ°ж–°еҖјзҡ„з®ҖеҚ•еҠЁз”»пјӣ

еҮ д№ҺжүҖжңүзҡ„еӣҫеұӮзҡ„еұһжҖ§йғҪжҳҜйҡҗжҖ§еҸҜеҠЁз”»зҡ„гҖӮдҪ еҸҜд»ҘеңЁж–ҮжЎЈдёӯзңӢеҲ°е®ғд»¬зҡ„з®Җд»ӢжҳҜд»Ҙ вҖҳanimatableвҖҷ з»“е°ҫзҡ„гҖӮиҝҷдёҚд»…еҢ…жӢ¬дәҶжҜ”еҰӮдҪҚзҪ®пјҢе°әеҜёпјҢйўңиүІжҲ–иҖ…йҖҸжҳҺеәҰиҝҷж ·зҡ„з»қеӨ§еӨҡж•°зҡ„ж•°еҖјеұһжҖ§пјҢз”ҡиҮід№ҹеӣҠжӢ¬дәҶеғҸ isHidden е’Ң doubleSided иҝҷж ·зҡ„еёғе°”еҖјгҖӮ еғҸ paths иҝҷж ·зҡ„еұһжҖ§д№ҹжҳҜ animatable зҡ„пјҢдҪҶжҳҜе®ғдёҚж”ҜжҢҒйҡҗејҸеҠЁз”»гҖӮ

1.2 CATransaction(жҳҫејҸ/йҡҗејҸдәӢеҠЎ)

CATransactionжҳҜCore Animationдёӯзҡ„дәӢеҠЎзұ»пјҢиҙҹиҙЈжү№йҮҸзҡ„жҠҠеӨҡдёӘеҜ№еӣҫеұӮж ‘(layer-tree)зҡ„дҝ®ж”№дҪңдёәдёҖдёӘеҺҹеӯҗжӣҙж–°еҲ°жёІжҹ“ж ‘гҖӮ

  • дәӢеҠЎжҳҜCore Animationз”ЁжқҘеҢ…еҗ«дёҖзі»еҲ—еұһжҖ§еҠЁз”»йӣҶеҗҲзҡ„жңәеҲ¶пјҢд»»дҪ•з”ЁжҢҮе®ҡдәӢеҠЎеҺ»ж”№еҸҳеҸҜеҠЁз”»зҡ„еӣҫеұӮеұһжҖ§йғҪдёҚдјҡз«ӢеҲ»еҸ‘з”ҹеҸҳеҢ–пјҢиҖҢжҳҜеҪ“дәӢеҠЎдёҖж—ҰжҸҗдәӨзҡ„ж—¶еҖҷејҖе§Ӣз”ЁдёҖдёӘеҠЁз”»иҝҮжёЎеҲ°ж–°еҖјгҖӮ
  • дәӢеҠЎжҳҜйҖҡиҝҮCATransactionзұ»жқҘеҒҡз®ЎзҗҶпјҢиҝҷдёӘзұ»зҡ„и®ҫи®ЎжңүдәӣеҘҮжҖӘпјҢдёҚеғҸдҪ д»Һе®ғзҡ„е‘ҪеҗҚйў„жңҹзҡ„йӮЈж ·еҺ»з®ЎзҗҶдёҖдёӘз®ҖеҚ•зҡ„дәӢеҠЎпјҢиҖҢжҳҜз®ЎзҗҶдәҶдёҖеҸ дҪ дёҚиғҪи®ҝй—®зҡ„дәӢеҠЎгҖӮCATransactionжІЎжңүеұһжҖ§жҲ–иҖ…е®һдҫӢж–№жі•пјҢе№¶дё”д№ҹдёҚиғҪз”Ё+allocе’Ң-initж–№жі•еҲӣе»әе®ғгҖӮиҖҢжҳҜз”Ёзұ»ж–№жі•+beginе’Ң+commitеҲҶеҲ«жқҘе…Ҙж ҲжҲ–иҖ…еҮәж ҲгҖӮ
  • ж”ҜжҢҒеөҢеҘ—дәӢеҠЎгҖӮ

еңЁiOSдёӯзҡ„еӣҫеұӮдёӯпјҢеҜ№еӣҫеұӮж ‘зҡ„жҜҸж¬Ўдҝ®ж”№йғҪеҝ…йЎ»жҳҜдәӢеҠЎзҡ„дёҖйғЁеҲҶгҖӮд»»дҪ•еҸҜеҠЁз”»зҡ„еӣҫеұӮеұһжҖ§пјҢеҸ‘з”ҹж”№еҸҳдә§з”ҹзҡ„еҠЁз”»йғҪдјҡиў«ж·»еҠ еҲ°ж ҲйЎ¶зҡ„дәӢеҠЎпјҢдҪ еҸҜд»ҘйҖҡиҝҮ+setAnimationDuration:ж–№жі•и®ҫзҪ®еҪ“еүҚдәӢеҠЎзҡ„еҠЁз”»ж—¶й—ҙпјҢжҲ–иҖ…йҖҡиҝҮ+animationDurationж–№жі•жқҘиҺ·еҸ–ж—¶й•ҝеҖјпјҲй»ҳи®Ө0.25з§’пјүгҖӮ

Core Animationж”ҜжҢҒдёӨз§Қзұ»еһӢзҡ„дәӢеҠЎпјҡйҡҗејҸдәӢеҠЎе’ҢжҳҫејҸдәӢеҠЎгҖӮ

  • еҪ“еӣҫеұӮж ‘иў«жІЎжңүжҳҫејҸдәӢеҠЎзҡ„зәҝзЁӢдҝ®ж”№ж—¶пјҢйҡҗејҸдәӢеҠЎдјҡиҮӘеҠЁеҲӣе»әпјҢе№¶еңЁзәҝзЁӢзҡ„ runloop дёӢдёҖж¬Ўиҝӯд»Јж—¶иҮӘеҠЁжҸҗдәӨгҖӮ
    • еҚіCore Animationдјҡзӣ‘жөӢдҝ®ж”№пјҢз„¶еҗҺеңЁжҜҸдёӘrunloopе‘ЁжңҹдёӯиҮӘеҠЁејҖе§ӢдёҖж¬Ўж–°зҡ„дәӢеҠЎпјҲrunloopжҳҜiOSиҙҹиҙЈж”¶йӣҶз”ЁжҲ·иҫ“е…ҘпјҢеӨ„зҗҶжңӘе®ҢжҲҗзҡ„е®ҡж—¶еҷЁжҲ–иҖ…зҪ‘з»ңдәӢд»¶пјҢжңҖз»ҲйҮҚж–°з»ҳеҲ¶еұҸе№•зҡ„дёңиҘҝпјүпјҢеҚідҪҝдҪ дёҚжҳҫејҸең°дҪҝз”Ё[CATransaction begin]ејҖе§ӢдёҖж¬ЎдәӢеҠЎпјҢеңЁдёҖдёӘзү№е®ҡrunloopеҫӘзҺҜдёӯзҡ„д»»дҪ•еұһжҖ§зҡ„еҸҳеҢ–йғҪдјҡиў«ж”¶йӣҶиө·жқҘпјҢз„¶еҗҺеҒҡдёҖж¬Ў0.25з§’зҡ„еҠЁз”»гҖӮ
  • еҪ“еә”з”ЁзЁӢеәҸеңЁдҝ®ж”№еӣҫеұӮж ‘д№ӢеүҚеҗ‘ CATransaction зұ»еҸ‘йҖҒ begin() ж¶ҲжҒҜпјҢз„¶еҗҺеҗ‘ CATransaction зұ»еҸ‘йҖҒ commit() ж¶ҲжҒҜж—¶пјҢе°ұдјҡеҸ‘з”ҹжҳҫејҸдәӢеҠЎгҖӮ

дәҢгҖҒйҡҗејҸеҠЁз”»

CoreAnimationж”ҜжҢҒдёӨз§Қзұ»еһӢзҡ„еҠЁз”»пјҡжҳҫејҸеҠЁз”»гҖҒйҡҗејҸеҠЁз”»гҖӮ

  • йҡҗејҸеҠЁз”»пјҡд№ӢжүҖд»ҘеҸ«йҡҗејҸпјҢжҳҜеӣ дёәжҲ‘д»¬е№¶жІЎжңүжҢҮе®ҡд»»дҪ•еҠЁз”»зҡ„зұ»еһӢгҖӮжҲ‘д»¬д»…д»…ж”№еҸҳдәҶдёҖдёӘеұһжҖ§пјҢз„¶еҗҺCore AnimationжқҘеҶіе®ҡеҰӮдҪ•е№¶дё”дҪ•ж—¶еҺ»еҒҡеҠЁз”»гҖӮ
  • жҳҫејҸеҠЁз”»пјҡ
    • йңҖиҰҒеҲӣе»әдёҖдёӘеҠЁз”»еҜ№иұЎпјҢе№¶и®ҫзҪ®ејҖе§Ӣе’Ңз»“жқҹеҖјпјҢзӣҙеҲ°жҠҠеҠЁз”»еә”з”ЁеҲ°жҹҗеӣҫеұӮдёҠпјҢеҠЁз”»жүҚејҖе§Ӣжү§иЎҢгҖӮ
    • жҳҫејҸеҠЁз”»ж—ўеҸҜд»ҘзӣҙжҺҘеҜ№еӣҫеұӮеұһжҖ§еҒҡеҠЁз”»пјҢд№ҹеҸҜд»ҘиҰҶзӣ–й»ҳи®Өзҡ„еӣҫеұӮиЎҢдёәгҖӮ

йҡҗејҸеҠЁз”»еә•еұӮжҳҜжҳҫејҸеҠЁз”»гҖӮпјҲиҜҰи§Ғ3.2иҠӮгҖҒ5.1.1иҠӮпјү

2.1 жј”зӨә

йҡҗејҸеҠЁз”»зңӢиө·жқҘиҝҷеӨӘжЈ’дәҶпјҢдјјд№ҺдёҚеӨӘзңҹе®һпјҢжҲ‘д»¬з”ЁдёҖдёӘdemoжқҘи§ЈйҮҠдёҖдёӢпјҡйҰ–е…Ҳе’Ңз¬¬дёҖз« вҖңеӣҫеұӮж ‘вҖқдёҖж ·еҲӣе»әдёҖдёӘи“қиүІзҡ„ж–№еқ—пјҢз„¶еҗҺж·»еҠ дёҖдёӘжҢүй’®пјҢйҡҸжңәж”№еҸҳе®ғзҡ„йўңиүІгҖӮзӮ№еҮ»жҢүй’®пјҢдҪ дјҡеҸ‘зҺ°еӣҫеұӮзҡ„йўңиүІе№іж»‘иҝҮжёЎеҲ°дёҖдёӘж–°еҖјпјҢиҖҢдёҚжҳҜи·іеҸҳгҖӮд»Јз ҒеҸҠжҳҫзӨәж•ҲжһңеҰӮдёӢпјҡ

жё…еҚ•7.1 йҡҸжңәж”№еҸҳеӣҫеұӮйўңиүІ

иҝҷе…¶е®һе°ұжҳҜжүҖи°“зҡ„йҡҗејҸеҠЁз”»гҖӮеҪ“дҪ ж”№еҸҳдёҖдёӘеұһжҖ§пјҢCore AnimationжҳҜеҰӮдҪ•еҲӨж–ӯеҠЁз”»зұ»еһӢе’ҢжҢҒз»ӯж—¶й—ҙзҡ„е‘ўпјҹе®һйҷ…дёҠеҠЁз”»жү§иЎҢзҡ„ж—¶й—ҙеҸ–еҶідәҺеҪ“еүҚдәӢеҠЎзҡ„и®ҫзҪ®пјҢеҠЁз”»зұ»еһӢеҸ–еҶідәҺеӣҫеұӮиЎҢдёә(action)гҖӮ

жҲ‘д»¬еҪ“з„¶еҸҜд»Ҙз”ЁеҪ“еүҚдәӢеҠЎзҡ„+setAnimationDuration:ж–№жі•жқҘдҝ®ж”№еҠЁз”»ж—¶й—ҙпјҢдҪҶеңЁиҝҷйҮҢжҲ‘д»¬йҰ–е…Ҳиө·дёҖдёӘж–°зҡ„дәӢеҠЎпјҢдәҺжҳҜдҝ®ж”№ж—¶й—ҙе°ұдёҚдјҡжңүеҲ«зҡ„еүҜдҪңз”ЁгҖӮеӣ дёәдҝ®ж”№еҪ“еүҚдәӢеҠЎзҡ„ж—¶й—ҙеҸҜиғҪдјҡеҜјиҮҙеҗҢдёҖж—¶еҲ»еҲ«зҡ„еҠЁз”»пјҲеҰӮеұҸе№•ж—ӢиҪ¬пјүпјҢжүҖд»ҘжңҖеҘҪиҝҳжҳҜеңЁи°ғж•ҙеҠЁз”»д№ӢеүҚеҺӢе…ҘдёҖдёӘж–°зҡ„дәӢеҠЎгҖӮ

дҝ®ж”№еҗҺзҡ„д»Јз Ғи§ҒдёӢж–№гҖӮиҝҗиЎҢзЁӢеәҸпјҢдҪ дјҡеҸ‘зҺ°иүІеқ—йўңиүІжҜ”д№ӢеүҚеҸҳеҫ—жӣҙж…ўдәҶгҖӮ

2.2 UIViewеҠЁз”»зҡ„еә•еұӮдәӢеҠЎ

еҰӮжһңдҪ з”ЁиҝҮUIViewзҡ„еҠЁз”»ж–№жі•еҒҡиҝҮдёҖдәӣеҠЁз”»ж•ҲжһңпјҢйӮЈд№Ҳеә”иҜҘеҜ№иҝҷдёӘжЁЎејҸдёҚйҷҢз”ҹгҖӮUIViewжңүдёӨдёӘж–№жі•пјҢ+beginAnimations:context:е’Ң+commitAnimationsпјҢе’ҢCATransactionзҡ„+beginе’Ң+commitж–№жі•зұ»дјјгҖӮе®һйҷ…дёҠеңЁ+beginAnimations:context:е’Ң+commitAnimationsд№Ӣй—ҙжүҖжңүи§ҶеӣҫжҲ–иҖ…еӣҫеұӮеұһжҖ§зҡ„ж”№еҸҳиҖҢеҒҡзҡ„еҠЁз”»йғҪжҳҜз”ұдәҺи®ҫзҪ®дәҶCATransactionзҡ„еҺҹеӣ гҖӮ

еңЁiOS4дёӯпјҢиӢ№жһңеҜ№UIViewж·»еҠ дәҶдёҖз§ҚеҹәдәҺblockзҡ„еҠЁз”»ж–№жі•пјҡ+animateWithDuration:animations:гҖӮиҝҷж ·еҶҷеҜ№еҒҡдёҖе Ҷзҡ„еұһжҖ§еҠЁз”»еңЁиҜӯжі•дёҠдјҡжӣҙеҠ з®ҖеҚ•пјҢдҪҶе®һиҙЁдёҠе®ғд»¬йғҪжҳҜеңЁеҒҡеҗҢж ·зҡ„дәӢжғ…гҖӮ

CATransactionзҡ„+beginе’Ң+commitж–№жі•еңЁ+animateWithDuration:animations:еҶ…йғЁиҮӘеҠЁи°ғз”ЁпјҢиҝҷж ·blockдёӯжүҖжңүеұһжҖ§зҡ„ж”№еҸҳйғҪдјҡиў«дәӢеҠЎжүҖеҢ…еҗ«гҖӮиҝҷж ·д№ҹеҸҜд»ҘйҒҝе…ҚејҖеҸ‘иҖ…з”ұдәҺеҜ№+beginе’Ң+commitеҢ№й…Қзҡ„еӨұиҜҜйҖ жҲҗзҡ„йЈҺйҷ©гҖӮ

2.3 дәӢеҠЎе®ҢжҲҗblock

3.2 CAAction

ж— и®әдҪ•ж—¶пјҢдёҖдёӘеҸҜеҠЁз”»зҡ„ layer еұһжҖ§ж”№еҸҳж—¶пјҢlayer йғҪдјҡеҜ»жүҫе№¶иҝҗиЎҢеҗҲйҖӮзҡ„ вҖҳactionвҖҷ жқҘе®һиЎҢиҝҷдёӘж”№еҸҳгҖӮеңЁ Core Animation зҡ„дё“дёҡжңҜиҜӯдёӯжҠҠиҝҷз§Қж”№еҸҳеұһжҖ§ж—¶CALayerиҮӘеҠЁеә”з”Ёзҡ„еҠЁз”»з§°дёәactionпјҢжҲ–иҖ… CAActionпјҢдёӯж–ҮиҜ‘дҪңеҠЁдҪңпјҢд№ҹз§°иЎҢдёәпјҲд»ҘдёӢз»ҹз§° иЎҢдёәпјүгҖӮ

CAAction(иЎҢдёә)йҖҡеёёжҳҜдёҖдёӘиў«Core AnimationйҡҗејҸи°ғз”Ёзҡ„жҳҫејҸеҠЁз”»еҜ№иұЎпјҲCAAnimation е®һзҺ°дәҶ<CAAction> еҚҸи®®пјүгҖӮ

3.2.1 CALayerдёҺCAActionеҚҸи®®

CAActionпјҡжҠҖжңҜдёҠжқҘиҜҙпјҢиҝҷжҳҜдёҖдёӘжҺҘеҸЈпјҢе№¶еҸҜд»Ҙз”ЁжқҘеҒҡеҗ„з§ҚдәӢжғ…гҖӮдҪҶжҳҜе®һйҷ…дёӯпјҢжҹҗз§ҚзЁӢеәҰдёҠдҪ еҸҜд»ҘеҸӘжҠҠе®ғзҗҶи§Јдёәз”ЁжқҘеӨ„зҗҶеҠЁз”»гҖӮ

жҳҜдёҖдёӘжҺҘеҸЈпјҢе…Ғи®ёеҜ№иұЎе“Қеә” CALayer ж”№еҸҳи§ҰеҸ‘зҡ„ actions

дёӢйқўжҳҜж‘ҳзҡ„CALayerдёӯпјҢжңүе…іCAActionзҡ„йғЁеҲҶеұһжҖ§гҖҒж–№жі•пјҡ

CALayer зҡ„ animatable еұһжҖ§йҖҡеёёйғҪе…·жңүзӣёеә”зҡ„ action object жқҘеҗҜеҠЁе®һйҷ…еҠЁз”»пјҢд№ҹе°ұжҳҜиҜҙеҪ“ CALayer ж”№еҸҳеұһжҖ§ж—¶дјҡиҮӘеҠЁеә”з”ЁеҠЁз”»гҖӮ

еҪ“CALayerзҡ„еұһжҖ§иў«дҝ®ж”№ж—¶еҖҷпјҢе®ғдјҡи°ғз”Ё-actionForKey:ж–№жі•пјҢдј йҖ’еұһжҖ§зҡ„еҗҚз§°гҖӮжқҘжҹҘжүҫеҲ°дёҺиҜҘеұһжҖ§еҗҚз§°е…іиҒ”зҡ„ action object пјҲйҒөе®Ҳ CAAction еҚҸи®®пјҢе№¶иғҪжҺҘж”¶ run(forKey:object:arguments:) ж¶ҲжҒҜпјүе№¶жү§иЎҢе®ғгҖӮ

жӮЁиҝҳеҸҜд»Ҙе°ҶиҮӘе®ҡд№ү action objects дёҺдҪ зҡ„ layer зӣёе…іиҒ”пјҢд»Ҙе®һзҺ°дёҖдәӣ APP зү№е®ҡзҡ„ж“ҚдҪңгҖӮ

жіЁж„ҸпјҡдёҠйқўзҡ„жӯҘйӘӨпјҢжҳҜеҜ№дәҺеҚ•зӢ¬зҡ„ layer жқҘиҜҙзҡ„гҖӮеҜ№дәҺ view дёӯзҡ„ layerпјҢеҜ№иЎҢдёәзҡ„жҗңзҙўеҸӘдјҡеҲ°з¬¬дёҖжӯҘдёәжӯўпјҲиҮіе°‘жҲ‘жІЎжңүи§ҒиҝҮ view иҝ”еӣһдёҖдёӘ nil з„¶еҗҺд»Қз„¶з»§з»ӯжҗңзҙўиЎҢдёәзҡ„жғ…еҶөпјүгҖӮ

еҜ№дёҖдёӘдёҖиҲ¬жқҘиҜҙеҸҜд»ҘеҠЁз”»зҡ„ layer еұһжҖ§еҗ‘ view иҜўй—®иЎҢдёәе°ұеҸҜд»ҘдәҶпјҢжҜ”еҰӮеҜ№дәҺ вҖҳpositionвҖҷпјҡ

иҝҗиЎҢдёҠйқўзҡ„д»Јз ҒпјҢеҸҜд»ҘзңӢеҲ°еңЁ block еӨ– view иҝ”еӣһзҡ„жҳҜ NSNull еҜ№иұЎпјҢиҖҢеңЁ block дёӯж—¶иҝ”еӣһзҡ„жҳҜдёҖдёӘ CABasicAnimationгҖӮеҫҲдјҳйӣ…пјҢеҜ№еҗ§пјҹеҖјеҫ—жіЁж„Ҹзҡ„жҳҜжү“еҚ°еҮәзҡ„ NSNull жҳҜеёҰзқҖдёҖеҜ№е°–жӢ¬еҸ·зҡ„ (вҖң<null>вҖң)пјҢиҝҷе’Ңе…¶д»–еҜ№иұЎдёҖж ·пјҢиҖҢжү“еҚ° nil зҡ„ж—¶еҖҷжҲ‘д»¬еҫ—еҲ°зҡ„жҳҜжҷ®йҖҡжӢ¬еҸ·((null))пјҡ

жҲ‘д»¬жқҘеҜ№йўңиүІжёҗеҸҳзҡ„дҫӢеӯҗдҪҝз”ЁдёҖдёӘдёҚеҗҢзҡ„иЎҢдёәпјҢйҖҡиҝҮз»ҷcolorLayerи®ҫзҪ®дёҖдёӘиҮӘе®ҡд№үзҡ„actionsеӯ—е…ёгҖӮжҲ‘д»¬д№ҹеҸҜд»ҘдҪҝз”Ёе§”жүҳжқҘе®һзҺ°пјҢдҪҶжҳҜactionsеӯ—е…ёеҸҜд»ҘеҶҷжӣҙе°‘зҡ„д»Јз ҒгҖӮйӮЈд№ҲеҲ°еә•ж”№еҰӮдҪ•еҲӣе»әдёҖдёӘеҗҲйҖӮзҡ„иЎҢдёәеҜ№иұЎе‘ўпјҹ

3.5 иҮӘе®ҡд№үеӣҫеұӮеұһжҖ§иЎҢдёә

иЎҢдёәйҖҡеёёжҳҜдёҖдёӘиў«Core AnimationйҡҗејҸи°ғз”Ёзҡ„жҳҫејҸеҠЁз”»еҜ№иұЎгҖӮиҝҷйҮҢжҲ‘д»¬дҪҝз”Ёзҡ„жҳҜдёҖдёӘе®һзҺ°дәҶCATransitionзҡ„е®һдҫӢпјҢеҸ«еҒҡжҺЁиҝӣиҝҮжёЎпјҲд»Јз ҒеҰӮдёӢпјүгҖӮ

жҳҫејҸеҠЁз”»дёӯзҡ„иҝҮжёЎпјҢдёҚеҶҚиөҳиҝ°пјҢеҜ№дәҺзҺ°еңЁпјҢеҸӘиҰҒзҹҘйҒ“CATransitionе“Қеә”CAActionеҚҸи®®пјҢе№¶дё”еҸҜд»ҘеҪ“еҒҡдёҖдёӘеӣҫеұӮиЎҢдёәе°ұи¶іеӨҹдәҶгҖӮз»“жһңеҫҲиөһпјҢдёҚи®әеңЁд»Җд№Ҳж—¶еҖҷж”№еҸҳиғҢжҷҜйўңиүІпјҢж–°зҡ„иүІеқ—йғҪжҳҜд»Һе·Ұдҫ§ж»‘е…ҘпјҢиҖҢдёҚжҳҜй»ҳи®Өзҡ„жёҗеҸҳж•ҲжһңгҖӮ

е®һзҺ°иҮӘе®ҡд№үиЎҢдёәпјҡ

дҪҝз”ЁжҺЁиҝӣиҝҮжёЎзҡ„иүІеҖјеҠЁз”»пјҡ

еӣӣгҖҒе‘ҲзҺ°еӣҫеұӮдёҺжЁЎеһӢеӣҫеұӮ

4.1 presentationLayerдёҺmodelLayer

CALayerзҡ„еұһжҖ§иЎҢдёәе…¶е®һеҫҲдёҚжӯЈеёёпјҢеӣ дёәж”№еҸҳдёҖдёӘеӣҫеұӮзҡ„еұһжҖ§е№¶жІЎжңүз«ӢеҲ»з”ҹж•ҲпјҢиҖҢжҳҜйҖҡиҝҮдёҖж®өж—¶й—ҙжёҗеҸҳжӣҙж–°гҖӮиҝҷжҳҜжҖҺд№ҲеҒҡеҲ°зҡ„е‘ўпјҹ

еҪ“дҪ ж”№еҸҳдёҖдёӘеӣҫеұӮзҡ„еұһжҖ§пјҢеұһжҖ§еҖјзҡ„зЎ®жҳҜз«ӢеҲ»жӣҙж–°зҡ„пјҲеҰӮжһңдҪ иҜ»еҸ–е®ғзҡ„ж•°жҚ®пјҢдҪ дјҡеҸ‘зҺ°е®ғзҡ„еҖјеңЁдҪ и®ҫзҪ®е®ғзҡ„йӮЈдёҖеҲ»е°ұе·Із»Ҹз”ҹж•ҲдәҶпјүпјҢдҪҶжҳҜеұҸе№•дёҠе№¶жІЎжңүй©¬дёҠеҸ‘з”ҹж”№еҸҳгҖӮиҝҷжҳҜеӣ дёәдҪ и®ҫзҪ®зҡ„еұһжҖ§е№¶жІЎжңүзӣҙжҺҘи°ғж•ҙеӣҫеұӮзҡ„еӨ–и§ӮпјҢзӣёеҸҚпјҢд»–еҸӘжҳҜе®ҡд№үдәҶеӣҫеұӮеҠЁз”»з»“жқҹд№ӢеҗҺе°ҶиҰҒеҸҳеҢ–зҡ„еӨ–и§ӮгҖӮ

еҪ“и®ҫзҪ®CALayerзҡ„еұһжҖ§пјҢе®һйҷ…дёҠжҳҜеңЁе®ҡд№үеҪ“еүҚдәӢеҠЎз»“жқҹд№ӢеҗҺеӣҫеұӮеҰӮдҪ•жҳҫзӨәзҡ„жЁЎеһӢгҖӮиҝҷйҮҢе°ұжҳҜдёҖдёӘе…ёеһӢзҡ„еҫ®еһӢMVCжЁЎејҸпјҡ

  • Core Animationжү®жј”дәҶдёҖдёӘжҺ§еҲ¶еҷЁзҡ„и§’иүІпјҢе№¶дё”иҙҹиҙЈж №жҚ®еӣҫеұӮиЎҢдёәе’ҢдәӢеҠЎи®ҫзҪ®еҺ»дёҚж–ӯжӣҙж–°и§Ҷеӣҫзҡ„иҝҷдәӣеұһжҖ§еңЁеұҸе№•дёҠзҡ„зҠ¶жҖҒгҖӮ
  • CALayerжҳҜдёҖдёӘиҝһжҺҘз”ЁжҲ·з•ҢйқўпјҲе°ұжҳҜMVCдёӯзҡ„viewпјүиҷҡжһ„зҡ„зұ»пјҢдҪҶжҳҜеңЁз•Ңйқўжң¬иә«иҝҷдёӘеңәжҷҜдёӢпјҢCALayerзҡ„иЎҢдёәжӣҙеғҸжҳҜеӯҳеӮЁдәҶи§ҶеӣҫеҰӮдҪ•жҳҫзӨәе’ҢеҠЁз”»зҡ„ж•°жҚ®жЁЎеһӢгҖӮ
  • е®һйҷ…дёҠпјҢеңЁиӢ№жһңиҮӘе·ұзҡ„ж–ҮжЎЈдёӯпјҢеӣҫеұӮж ‘йҖҡеёёйғҪжҳҜжҢҮзҡ„еӣҫеұӮж ‘жЁЎеһӢгҖӮ

еңЁiOSдёӯпјҢеұҸе№•жҜҸз§’й’ҹйҮҚз»ҳ60ж¬ЎгҖӮеҰӮжһңеҠЁз”»ж—¶й•ҝжҜ”60еҲҶд№ӢдёҖз§’иҰҒй•ҝпјҢCore Animationе°ұйңҖиҰҒеңЁи®ҫзҪ®дёҖж¬Ўж–°еҖје’Ңж–°еҖјз”ҹж•Ҳд№Ӣй—ҙпјҢеҜ№еұҸе№•дёҠзҡ„еӣҫеұӮиҝӣиЎҢйҮҚж–°з»„з»ҮгҖӮиҝҷж„Ҹе‘ізқҖCALayerйҷӨдәҶвҖңзңҹе®һвҖқеҖјпјҲе°ұжҳҜдҪ и®ҫзҪ®зҡ„еҖјпјүд№ӢеӨ–пјҢеҝ…йЎ»иҰҒзҹҘйҒ“еҪ“еүҚжҳҫзӨәеңЁеұҸе№•дёҠзҡ„еұһжҖ§еҖјзҡ„и®°еҪ•гҖӮ

жҜҸдёӘеӣҫеұӮеұһжҖ§зҡ„жҳҫзӨәеҖјйғҪиў«еӯҳеӮЁеңЁдёҖдёӘеҸ«еҒҡе‘ҲзҺ°еӣҫеұӮзҡ„зӢ¬з«ӢеӣҫеұӮеҪ“дёӯпјҢд»–еҸҜд»ҘйҖҡиҝҮ-presentationLayerж–№жі•жқҘи®ҝй—®гҖӮиҝҷдёӘе‘ҲзҺ°еӣҫеұӮе®һйҷ…дёҠжҳҜжЁЎеһӢеӣҫеұӮзҡ„еӨҚеҲ¶пјҢдҪҶжҳҜе®ғзҡ„еұһжҖ§еҖјд»ЈиЎЁдәҶеңЁд»»дҪ•жҢҮе®ҡж—¶еҲ»еҪ“еүҚеӨ–и§Ӯж•ҲжһңгҖӮжҚўеҸҘиҜқиҜҙпјҢдҪ еҸҜд»ҘйҖҡиҝҮе‘ҲзҺ°еӣҫеұӮзҡ„еҖјжқҘиҺ·еҸ–еҪ“еүҚеұҸе№•дёҠзңҹжӯЈжҳҫзӨәеҮәжқҘзҡ„еҖјгҖӮ

еҰӮеӣҫпјҢдёҖдёӘз§»еҠЁзҡ„еӣҫеұӮжҳҜеҰӮдҪ•йҖҡиҝҮж•°жҚ®жЁЎеһӢе‘ҲзҺ°зҡ„пјҡ

жҲ‘д»¬еңЁжң¬д№Ұзҡ„з¬¬дёҖз« дёӯжҸҗеҲ°йҷӨдәҶеӣҫеұӮж ‘пјҢеҸҰеӨ–иҝҳжңүе‘ҲзҺ°ж ‘гҖӮе‘ҲзҺ°ж ‘йҖҡиҝҮеӣҫеұӮж ‘дёӯжүҖжңүеӣҫеұӮзҡ„е‘ҲзҺ°еӣҫеұӮжүҖеҪўжҲҗгҖӮжіЁж„Ҹе‘ҲзҺ°еӣҫеұӮд»…д»…еҪ“еӣҫеұӮйҰ–ж¬Ўиў«жҸҗдәӨпјҲе°ұжҳҜйҰ–ж¬Ўз¬¬дёҖж¬ЎеңЁеұҸе№•дёҠжҳҫзӨәпјүзҡ„ж—¶еҖҷеҲӣе»әпјҢжүҖд»ҘеңЁйӮЈд№ӢеүҚи°ғз”Ё-presentationLayerе°Ҷдјҡиҝ”еӣһnilгҖӮ

дҪ еҸҜиғҪжіЁж„ҸеҲ°жңүдёҖдёӘеҸ«еҒҡвҖ“modelLayerзҡ„ж–№жі•гҖӮеңЁе‘ҲзҺ°еӣҫеұӮдёҠи°ғз”ЁвҖ“modelLayerе°Ҷдјҡиҝ”еӣһе®ғжӯЈеңЁе‘ҲзҺ°жүҖдҫқиө–зҡ„CALayerгҖӮйҖҡеёёеңЁдёҖдёӘеӣҫеұӮдёҠи°ғз”Ё-modelLayerдјҡиҝ”еӣһвҖ“selfпјҲе®һйҷ…дёҠжҲ‘д»¬е·Із»ҸеҲӣе»әзҡ„еҺҹе§ӢеӣҫеұӮе°ұжҳҜдёҖз§Қж•°жҚ®жЁЎеһӢпјүгҖӮ

4.2 е‘ҲзҺ°еӣҫеұӮзҡ„дҪҝз”ЁеңәжҷҜ

еӨ§еӨҡж•°жғ…еҶөдёӢпјҢдҪ дёҚйңҖиҰҒзӣҙжҺҘи®ҝй—®е‘ҲзҺ°еӣҫеұӮпјҢдҪ еҸҜд»ҘйҖҡиҝҮе’ҢжЁЎеһӢеӣҫеұӮзҡ„дәӨдә’пјҢжқҘи®©Core Animationжӣҙж–°жҳҫзӨәгҖӮдёӨз§Қжғ…еҶөдёӢе‘ҲзҺ°еӣҫеұӮдјҡеҸҳеҫ—еҫҲжңүз”ЁпјҢдёҖдёӘжҳҜеҗҢжӯҘеҠЁз”»пјҢдёҖдёӘжҳҜеӨ„зҗҶз”ЁжҲ·дәӨдә’гҖӮ

еҰӮжһңжҲ‘д»¬жғіиҰҒеҒҡзҡ„дёҚиҝҮжҳҜж·»еҠ дёҖдёӘд»Һж—§зҡ„еҖјеҗ‘ж–°зҡ„еҖјиҝҮеәҰзҡ„еҠЁз”»зҡ„иҜқпјҢжҲ‘д»¬еҸҜд»ҘзӣҙжҺҘеңЁ delegate зҡ„еӣһи°ғдёӯжқҘеҒҡгҖӮз„¶иҖҢеӣ дёәжҲ‘д»¬жғіиҰҒжӣҙзІҫзЎ®ең°жҺ§еҲ¶еҠЁз”»пјҢжҲ‘д»¬йңҖиҰҒз”ЁдёҖдёӘеё§еҠЁз”»жқҘе®һзҺ°гҖӮеё§еҠЁз”»йңҖиҰҒжүҖжңүзҡ„еҖјйғҪжҳҜе·ІзҹҘзҡ„пјҢиҖҢеҜ№жҲ‘д»¬зҡ„жғ…еҶөжқҘиҜҙпјҢж–°зҡ„еҖјиҝҳжІЎжңүиў«и®ҫе®ҡпјҢеӣ жӯӨжҲ‘д»¬д№ҹе°ұж— д»ҺзҹҘжҷ“гҖӮ

жңүж„ҸжҖқзҡ„жҳҜпјҢiOS ж·»еҠ зҡ„дёҖдёӘеҹәдәҺ block зҡ„еҠЁз”» API д№ҹйҒҮеҲ°дәҶеҗҢж ·зҡ„й—®йўҳгҖӮдҪҝз”Ёе’ҢдёҠйқўдёҖж ·зҡ„и§ӮеҜҹжүӢж®өпјҢжҲ‘д»¬е°ұиғҪзҹҘйҒ“е®ғжҳҜеҰӮдҪ•з»•ејҖиҝҷдёӘйә»зғҰзҡ„гҖӮеҜ№дәҺжҜҸдёӘе…ій”®её§пјҢеңЁеұһжҖ§еҸҳеҢ–ж—¶пјҢview иҝ”еӣһ nilпјҢдҪҶжҳҜеҚҙеӯҳеӮЁдёӢйңҖиҰҒзҡ„зҠ¶жҖҒгҖӮиҝҷж ·е°ұиғҪеңЁжүҖжңүе…ій”®её§ block жү§иЎҢеҗҺеҲӣе»әдёҖдёӘ CAKeyframeAnimation еҜ№иұЎгҖӮ

4. еҲӣе»әе…ій”®её§еҠЁз”»
